Are You College Material?

Project Overview

College Enrollment Analysis and Graduate School Asmissions Predictor. In this project we provide an oveview of college enrollment data during a particular year as provided in the dataset. We Will also train regression models to find the probability of being accepted into graduate school in the United States.

In this project we will go through the following tasks:

(1) Understand the Problem Statement
(2) Import libraries and datasets
(3) Perform Exploratory Data Analysis
(4) Perform Data Visualization
(5) Create Training and Testing Datasets
(6) Train and Evaluate a Linear Regression Model

Tools Used: Python Pandas, Flask, Python Matplotlib, HTML/CSS/Bootstrap, JavaScript, JQuery, Tableau, and Heroku
